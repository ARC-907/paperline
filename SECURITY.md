# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities via **GitHub Security Advisories**:
https://github.com/ARC-907/paperline/security/advisories/new

We aim to acknowledge reports within 5 business days and to provide a fix or mitigation timeline within 14 business days.

## Supported Versions

Only the latest `main` branch is supported with security updates at this time.

## Scope

In-scope:
- Code, configuration, and CI workflows shipped from this repository.
- Documentation that could mislead a user into an insecure configuration.

Out of scope:
- Vulnerabilities in third-party dependencies (please report upstream).
- Social-engineering against project maintainers.
- Issues that require a compromised host or non-default configuration.
